# ES 204 Memristor ODE Project

This repository compares Explicit Euler, RK4, and Implicit Euler on a bounded
HP TiO2 memristor ODE. The main goal is to study timestep sensitivity,
boundary overshoot, V-I hysteresis loops, and runtime cost.

## Main Files

- `Lagramada-Villareal-Proposal.pdf` - original project proposal.
- `src/memristor_methods.ipynb` - main executable notebook.
- `Agent.md` - implementation notes and solver details.
- `latex/report/report.pdf` - short report.
- `latex/exhaustive_report/exhaustive_report.pdf` - longer presentation report.
- `latex/figures/` - figures generated by the notebook.
- `latex/tables/` - CSV summary tables generated by the notebook.
- `notes/` - course notes used for the numerical methods.

## Setup With uv

From the repository root:

```bash
uv sync
```

This creates or updates `.venv/` using the dependencies in `pyproject.toml`
and `uv.lock`.

The project uses:

- NumPy
- Pandas
- Matplotlib
- Jupyter

## Install the Jupyter Kernel

Install the virtual environment as a Jupyter kernel:

```bash
uv run python -m ipykernel install --user --name es204 --display-name "ES204 Project"
```

Check that it was installed:

```bash
uv run jupyter kernelspec list
```

You should see a kernel named `es204` with display name `ES204 Project`.

## Run the Notebook

To open Jupyter Lab:

```bash
uv run jupyter lab
```

Then open:

```text
src/memristor_methods.ipynb
```

Select the `ES204 Project` kernel if it is not already selected.

To execute the notebook from the terminal:

```bash
uv run jupyter nbconvert \
  --to notebook \
  --execute \
  --inplace src/memristor_methods.ipynb \
  --ExecutePreprocessor.kernel_name=es204 \
  --ExecutePreprocessor.timeout=3600
```

Running the notebook regenerates:

- `latex/figures/state_trajectories.png`
- `latex/figures/bound_violations.png`
- `latex/figures/hysteresis_loops.png`
- `latex/figures/runtime_per_step.png`
- `latex/tables/overshoot_summary.csv`
- `latex/tables/final_state_summary.csv`
- `latex/tables/runtime_summary.csv`

## Compile the Reports

Short report:

```bash
cd latex/report
pdflatex report.tex
pdflatex report.tex
```

Exhaustive report:

```bash
cd latex/exhaustive_report
pdflatex exhaustive_report.tex
pdflatex exhaustive_report.tex
```

Run `pdflatex` twice so references and the table of contents update correctly.

## Important Interpretation Note

The raw state plots are a stress test of unconstrained numerical methods.
A coarse timestep can sometimes look better only because it misses important
parts of the sinusoidal forcing. A bounded-looking raw curve is not
automatically more accurate.

For physical V-I plots, the notebook uses clipped states. Raw states are still
kept separately so boundary overshoot can be measured.
